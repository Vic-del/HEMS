# High-End Mobile Searcher

_Retrieves the best mobiles from antutu and deploys their mean price on internet._

## Exec

* Install [scrapy](https://scrapy.org/)
* Pull repository
* Use commands: scrapy crawl mobiles & scrapy crawl prices
